# Paint

Fabric.js를 활용하여 간단한 도형 그림판을 만들었습니다.

express + socket.io를 활용하여 채팅 기능을 추가하려 합니다.

궁극적으로 그림판을 공유하고 채팅을 할 수 있는 웹을 만드려 합니다.
